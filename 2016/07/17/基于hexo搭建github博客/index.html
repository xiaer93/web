基于hexo搭建github博客

Gitpage，你值得拥有！

一直有搭建个人博客的想法，一是学习网页编程，二是为了显摆o_O。最开始以为要购买云服务器才能搭建个人博客，要学会Html/CSS，还有MySql数据库。可是这些都太难了，前不久偶然看到Github提供Gitpage服务，可以用来部署个人博客，经过搜索搜索，最后选择利用Hexo搭建博客，部署在Github。经过折腾，果然成功啦，哇咔咔，以后常写写博客，万一有什么奇妙用处。

开发环境

PS:我的电脑是win7 64位操作系统。

  1. nodejs。。。安装nodejs最新的版本，老版本可能会存在各种问题，进入nodejs中文官网，下载相关软件。
  2. Git 。。。安装git软件，用于在github部署博客，进入git，下载相关软件。
  3. Hexo 。。。安装Hexo软件，在Git Bash（命令行）输入npm install -g hexo，系统自动安装Hexo。

强烈建议以下操作在git bash里进行。不要在cmd里敲git命令！

链接Github（SSH Key）

  1. 设置Git的user.name和user.email

    在第一次使用Git时，Git需要两个信息。具体通过以下命令设置：

    git config --global user.name "voidking"

    git config --global user.email "voidking@qq.com"

  2. 生成SSH密钥

    在Git Bash中输入以下指令（邮箱就是你注册Github的邮箱），回车（回车后要求输入密码，可以无视直接回车）：

    ssh-keygen -t rsa -C "angelen10@163.com"

  3. 复制SSH公钥

    输入clip < ~/.ssh/id_rsa.pub，回车后公钥就被拷贝到剪切板了，可以直接粘贴使用。

  4. 在Github添加SSH密钥

    在Github网站中，选择Settings，点击SSH and GPG keys，接着点击点击Add SSH key，在key中粘贴即可。

  5. 测试
    ssh -T git@github.com，回车后如有警告输入’yes’，最终提示：Hi xiaer93！You’ve successfully authenticated

搭建Blog

  1. 创建hexo文件夹

    在任意位置下（如c:/blog），打开Git Bash，执行命令：hexo init,Hexo会自动在该文件夹下建立相关网站文件。

  2. 安装依赖包

    npm install

  3. 本地查看

    hexo g

    hexo s

    打开浏览器输入localhost:4000查看网站效果，至此，本地Blog已经搭建好了。

部署Blog到Github

  1. 创建仓库

    在github官网上创建仓库，仓库名与github用户名相同。例如我的github用户名为xiaer93，仓库名为xiaer93.github.io。

  2. 修改_config.yml

    编辑c:/blog下的_config.yml，修改Deployment部分，重要，重要，重要，请按照下述更改_config.yml：

    deploy:

    `type: git`

`repository: git@github.com:xiaer93/xiaer93.github.io.git`

`branch: master`

  3. 执行命令：

    npm install hexo-deployer-git --save

  4. 开始部署：

    hexo d

  5. 访问测试：

    http://xiaer93.github.io

恭喜，你已经成功啦，那就开始写博客吧^_^
