科学上网神器shadowsocks

我希望有一天我能够住在一个拥有随意编写任何我喜欢的代码的国家而无需为之恐惧。

我希望你们将来能为互联网自由贡献出了不起的东西。


一直听说我们有个墙，也有个饭墙的冲动，谁让哪里有压迫哪里就有反抗呢？之前一直用的“自由门”软件，但是感觉很不稳定，大多数时间用不。折腾了好久，找各种饭墙软件啦，还有VPN等等，但是都感觉有点不靠谱，网友评论也是各种各种。直到最近发现了Shadowsocks，瞬间感觉有希望了，查看维基百科，居然还能有这么心酸的故事。

Shadowsocks（中文名称：影梭）是使用Python等语言开发的、基于Apache许可证开源的代理软件。Shadowsocks使用socks5代理，用于保护网络流量。在中国大陆被广泛用于突破防火长城（GFW），以浏览被封锁的内容。

Shadowsocks分为服务器端和客户端。在使用之前，需要先将服务器端部署在支持Python的服务器上面，然后通过客户端连接并创建本地代理。

2015年8月22日，Shadowsocks原作者Clowwindy因受到中国政府的压力，停止维护此项目。但开发已有人接管，仍有更新陆续推送。ShadowsocksR可作为Shadowsocks的替代软件。

基于hexo搭建github博客

Gitpage，你值得拥有！

一直有搭建个人博客的想法，一是学习网页编程，二是为了显摆o_O。最开始以为要购买云服务器才能搭建个人博客，要学会Html/CSS，还有MySql数据库。可是这些都太难了，前不久偶然看到Github提供Gitpage服务，可以用来部署个人博客，经过搜索搜索，最后选择利用Hexo搭建博客，部署在Github。经过折腾，果然成功啦，哇咔咔，以后常写写博客，万一有什么奇妙用处。

开发环境

PS:我的电脑是win7 64位操作系统。

  1. nodejs。。。安装nodejs最新的版本，老版本可能会存在各种问题，进入nodejs中文官网，下载相关软件。
  2. Git 。。。安装git软件，用于在github部署博客，进入git，下载相关软件。
  3. Hexo 。。。安装Hexo软件，在Git Bash（命令行）输入npm install -g hexo，系统自动安装Hexo。
