landscape主题修改

hexo原生主题看起来不是很满意，因此对其进行了一些修改！

主要修改

  • layout_partial，这里为局部模版区域，修改html代码
  • source\css，这里为样式区域，修改css代码
  • source\js，这里为javascript区域，修改js代码
  • _config.yml，这是主题配置信息

1. 新增自定义菜单works，支持展示个人作品

  • 使用hexo3.0数据文件功能，通过project.json传入作品集信息！

    [
    {
        "name":"jq插件之日期选择器",
        "date":"2017-5",
        "url":"#",
        "img":"http://o7bkkhiex.bkt.clouddn.com/item-apple.jpg",
        "title":"jq插件",
        "description":"jq插件实现的日历选择器"
    }
]

  • 添加works.ejs模版

    <% site.data.project.forEach(function (work,index) {  %>
<article class="work-inner">
  <header class="clear-float">
    <h3><%= work.name %></h3>
    <address><%= work.date%></address>
  </header>
  <img src="<%= work.img%>" alt="work.title">
  <div class="description">
    <h4><%= work.title%></h4>
    <p><%= work.description%></p>
  </div>
  <footer>
    <a class="work_link" href="<%= work.url%>">Open it！</a>
  </footer>
</article>
<% }) %>

  • 添加自定义菜单

    在命令行里面输入：hexo new page "about"

然后你会发现source里面多了个目录about，里面有个index.md。

接着把链接加上，themes/<theme_name>/_config.yml里面的menu一项，添加一行About: /about。

  • 参考链接：

    • hexo模版之Anisina
    • hexo怎么在菜单添加页面和分类

2. 添加百度统计

  • 直接阅读官方教程，将百度统计的相关js添加至html文档中！安装案例

3. 添加不蒜子

  • 不蒜子用于统计网站浏览量，并直接显示出来。直接阅读官网教程，添加不蒜子功能
