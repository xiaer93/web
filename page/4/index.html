关于提升产品质量的思考

质量于企业来说，是非常非常重要的，一是对自己负责，二是对客户负责。随着科学技术的发展，各种工业设备在方便社会生产的同时，也潜在的巨大的危险。所以，重视产品质量，显得尤为的重要，本文仅扯淡关于我的思考。

质量管理

人机物法环，人当先，质量在于人。

质量定义

在ISO9001中,对质量有如下定义，“一组固有特性满足要求的程度”，而我的见解比较狭义，就是要对客户负责，保证功能，保障安全。

质量体系

现代企业发展数百年里，涌现出各种优秀的质量管理体系，如ISO9001、6S、八到位、精益制造等等理论，其核心是希望通过规范标准的程序方法引导全体成员参与质量管理，重视产品质量。通过各种体系方法，让产品质量能够落地生根，变为员工切身可以去做的事情，让每个人都是质量的参与者、监督者和保证者。但是，鉴于中国的传统文化思想，多数管理体系没有得到有效的落实，保证产品质量也就变成了一个口号。人机物法环，质量在人，如何解决管理体系的落实难题呢？

基于hexo搭建github博客

Gitpage，你值得拥有！

一直有搭建个人博客的想法，一是学习网页编程，二是为了显摆o_O。最开始以为要购买云服务器才能搭建个人博客，要学会Html/CSS，还有MySql数据库。可是这些都太难了，前不久偶然看到Github提供Gitpage服务，可以用来部署个人博客，经过搜索搜索，最后选择利用Hexo搭建博客，部署在Github。经过折腾，果然成功啦，哇咔咔，以后常写写博客，万一有什么奇妙用处。

开发环境

PS:我的电脑是win7 64位操作系统。

  1. nodejs。。。安装nodejs最新的版本，老版本可能会存在各种问题，进入nodejs中文官网，下载相关软件。
  2. Git 。。。安装git软件，用于在github部署博客，进入git，下载相关软件。
  3. Hexo 。。。安装Hexo软件，在Git Bash（命令行）输入npm install -g hexo，系统自动安装Hexo。
